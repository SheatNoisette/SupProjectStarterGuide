    Liens utiles

    Voici une liste, non exhaustive, de liens pouvant vous être utiles pour votre projet de jeu vidéo et tout ce qui tourne autour.

    • Sites web des projets de S2 des promos précédentes (ici 2023) : epita.it/2023/
    • Site web d'Unity : unity.com
    • Documentation officielle d'Unity (très utile) : docs.unity3d.com
    • Overleaf (éditeur LaTeX en ligne) : fr.overleaf.com
    • Télécharger TeX Live (pour faire du LaTeX en local) : tug.org
    • GitHub.com et GitLab.com qu'on ne présente plus
    • Chaîne YouTube de Brackeys (Chaîne YouTube inactive), fournissant de nombreux tutoriels Unity : www.youtube.com/brackeys
    • Photon Engine, qui propose Photon Unity Networking (à prendre en v2 !), un SDK de multijoueur en ligne simple d'emploi : photonengine.com
    • Mirror Networking, une alternative Open Source à Photon (Plus adapté aux MMOs): mirror-networking.com
    • Une fiche de rappel sur les commandes les plus utiles de Git, le tout avec des illustrations : GitLab Git Cheatsheet
