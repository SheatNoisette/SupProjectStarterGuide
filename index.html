    Projet Informatique : Guide de survie

    Ce site est un guide permettant de surmonter divers problèmes rencontrés lors de la réalisation d'un projet de jeu vidéo sous Unity3D. D'autres sujets sont abordés tels que LaTeX, Markdown, Git ou encore Vim.

    Ce guide fut initialement écrit pour le projet informatique du S2 d'Epita mais peut être adapté à d'autres projets scolaires de jeu vidéo fait avec Unity et impliquant également l'utilisation de Git et la rédaction de rapports avec LaTeX.

    Auteurs

    Benoît “SheatNoisette” M. et Luca “iralorucrie” Corrieri.

    Remerciements particuliers à EPITA.it pour fournir gracieusement le sous-domaine s2guide.epita.it.
